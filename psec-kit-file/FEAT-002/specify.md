# Specification: Event Listing (FEAT-002)

## المتطلبات الوظيفية
- عرض قائمة فعاليات جارية ومستقبلية.
- لكل فعالية تظهر المعلومات التالية:
  - اسم الفعالية
  - التاريخ
  - الوقت
  - المكان
  - التصنيف
  - وصف مختصر
  - اسم الجهة المنظمة
- ترتيب الفعاليات حسب التاريخ (الأحدث أولاً).
- دعم اللغة العربية في جميع العناصر.
- واجهة متجاوبة وسريعة التحميل.

## المتطلبات غير الوظيفية
- الأداء: تحميل القائمة خلال أقل من 2 ثانية في ظروف الإنترنت الضعيف.
- الأمان: لا يتم عرض بيانات حساسة.
- التوافق: يعمل على جميع الأجهزة (موبايل وكمبيوتر).

## API
- Endpoint: GET /api/events/list
- الاستجابة:
```json
[
  {
    "id": 1,
    "name": "فعالية تقنية دمشق",
    "date": "2024-07-10",
    "time": "10:00",
    "location": "دمشق، المركز الثقافي",
    "category": "تقني",
    "description": "ملتقى تقني للمبرمجين في دمشق.",
    "organizer": "جمعية البرمجيات السورية"
  },
  ...
]
```

## واجهة المستخدم
- تصميم بسيط، بطاقات أو جدول فعاليات.
- دعم RTL للغة العربية.
