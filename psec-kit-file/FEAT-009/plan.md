# Plan: FEAT-009 - Organizer Information Display

## Backend (C#, ABP)
- تحديث نموذج بيانات الفعالية ليشمل:
  - اسم الجهة المنظمة (OrganizerName)
  - اسم مسؤول التواصل (ContactPersonName)
  - وسيلة التواصل (ContactEmail, ContactPhone)
- التأكد من أن الحقول إلزامية عند إنشاء الفعالية.
- تضمين هذه الحقول في API استرجاع تفاصيل الفعالية.

## Frontend (Angular)
- عرض اسم الجهة المنظمة ومسؤول التواصل بشكل بارز في صفحة تفاصيل الفعالية.
- عرض وسيلة التواصل (بريد إلكتروني/هاتف) مع أيقونة مناسبة.
- دعم كامل للغة العربية في العرض.
- التأكد من تناسق التصميم مع باقي عناصر صفحة الفعالية.

## Testing
- اختبار ظهور المعلومات بشكل صحيح في جميع الحالات.
- اختبار التوافق مع الأجهزة المحمولة.
