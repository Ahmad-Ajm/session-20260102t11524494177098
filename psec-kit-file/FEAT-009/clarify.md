# Clarification: FEAT-009 - Organizer Information Display

## Context
ميزة عرض معلومات المنظم ومسؤول التواصل على صفحات الفعالية تهدف إلى تعزيز الشفافية والثقة للمشاركين، وتسهيل التواصل مع المنظمين.

## Requirements
- يجب عرض اسم الجهة المنظمة بشكل واضح في صفحة تفاصيل الفعالية.
- يجب عرض اسم مسؤول التواصل مع وسيلة تواصل (بريد إلكتروني أو رقم هاتف).
- المعلومات تظهر باللغتين (العربية أولاً).
- يجب أن تكون المعلومات جزءاً من نموذج بيانات الفعالية في الباك إند.
- لا حاجة لإمكانيات تعديل من الواجهة الأمامية في النسخة الأولى (المعلومات تضاف عند إنشاء الفعالية فقط).

## Assumptions
- الجهة المنظمة ومسؤول التواصل هما حقول إلزامية عند إضافة الفعالية.
- لا يوجد تحقق متقدم من صحة بيانات التواصل في النسخة الأولى.
- لا يوجد نظام مراسلة داخلي حالياً.

## Dependencies
- تعتمد هذه الميزة على وجود صفحة تفاصيل الفعالية (FEAT-003).
